# 🏛️ Tegal Culture - Warisan Budaya Kota Bahari

Tegal Culture adalah sebuah platform digital berbasis web yang dirancang untuk mengeksplorasi, mempromosikan, dan melestarikan identitas budaya Kota Tegal. Mulai dari situs sejarah, kemeriahan event lokal, hingga potensi UMKM unggulan.

## 🚀 Fitur Utama

### 👤 Dashboard User (Public)
- **Jelajah Budaya**: Eksplorasi tempat bersejarah dengan kategori (Wisata, Sejarah, Tradisi, Kuliner).
- **Event Budaya**: Daftar acara budaya mendatang lengkap dengan lokasi dan waktu.
- **UMKM Unggulan**: Katalog produk lokal (Fashion, Kuliner, Kriya) untuk mendukung ekonomi kreatif.
- **Fitur Pencarian Real-time**: Mencari destinasi atau produk secara instan tanpa reload halaman.
- **Filter Kategori Dinamis**: Menyaring konten berdasarkan minat pengguna dengan animasi yang halus.
- **UI/UX Premium**: Desain modern dengan efek *Glassmorphism*, animasi *AOS*, dan *bounce effects*.

### 🔑 Dashboard Admin
- **Manajemen Konten**: Akses penuh (CRUD) untuk menambah, mengubah, dan menghapus data Jelajah Budaya, Event, dan UMKM.
- **Upload Gambar**: Sistem manajemen media untuk aset visual tempat dan produk.
- **Statistik Ringkas**: Pantau jumlah lokasi dan data yang terinput.

## 🛠️ Tech Stack

- **Backend**: [Python](https://www.python.org/) & [Flask Framework](https://flask.palletsprojects.com/)
- **Database**: [SQLAlchemy]([https://www.sqlalchemy.org/](https://supabase.com/)) Build in a weekendScale to millions
- **Frontend**: [Jinja2](https://jinja.palletsprojects.com/) (Templating Engine)
- **Styling**: [Bootstrap 5](https://getbootstrap.com/), CSS3 Kustom
- **Animation**: [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/) & JavaScript ES6
