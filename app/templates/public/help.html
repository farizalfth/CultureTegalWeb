Kategori A: Pendaftaran, OTP & Google Sign-In (Supabase Auth)

Saat mendaftar menggunakan email dan kata sandi, sistem keamanan kami (didukung oleh Supabase Auth) akan mengirimkan kode verifikasi OTP (One-Time Password) ke email Anda. Masukkan kode tersebut pada halaman verifikasi. Kode OTP ini memiliki batas waktu kedaluwarsa maksimal 15 menit. Jika Anda tidak menerimanya, silakan periksa folder "Spam" atau "Promosi". Jika dalam 60 detik kode belum masuk, Anda dapat mengetuk tombol "Kirim ulang" setelah penghitung waktu mundur selesai.

Fitur "Daftar/Masuk dengan Google" membutuhkan komunikasi rill antara Google Play Services di perangkat seluler Anda dengan server otentikasi. Kendala ini biasanya terjadi jika koneksi internet Anda tidak stabil, Google Play Services di ponsel Anda belum diperbarui, atau sertifikat tanda tangan rilis (SHA-1 Keystore) sedang mengalami proses pembaruan berkala di Google Play Console. Pastikan jaringan internet Anda lancar dan coba kembali.

Setelah Anda berhasil memverifikasi email melalui Supabase, sistem seluler akan mengirimkan token akses ke server backend Flask kami via endpoint "/auth/sync" untuk mendaftarkan nama, email, dan mengatur ID notifikasi OneSignal Anda ke database utama. Jika proses sinkronisasi ini terganggu oleh jaringan, Anda mungkin akan diminta untuk melakukan masuk ulang (*re-login*) demi memastikan data profil Anda tersinkronisasi sempurna di server.

Pesan ini muncul jika Anda mengklik tautan pemulihan kata sandi (*Password Reset Link*) yang dikirimkan ke email Anda setelah masa berlakunya habis (melebihi batas waktu kedaluwarsa tautan keamanan), atau jika tautan tersebut sudah pernah diklik sebelumnya. Silakan masuk kembali ke menu "Lupa kata sandi?" di aplikasi untuk meminta pengiriman kode OTP pemulihan yang baru.

Kategori B: Fitur Pindai AI Kuliner (YOLO26 Nano)

Aplikasi kami menggunakan model jaringan saraf tiruan super-ringan (YOLO26 Nano) yang dirancang khusus untuk berjalan secara efisien di server Flask. Karena kami memprioritaskan keringan beban komputasi agar baterai ponsel Anda tidak boros, model ini dilatih menggunakan dataset gambar kuliner khas lokal yang terkelola secara terbatas. Akurasi deteksi ini bersifat probabilistik (perkiraan) dan sangat dipengaruhi oleh kecerahan ruangan, bayangan, serta sudut pengambilan gambar makanan.

Demi kenyamanan visual antarmuka pengguna di layar ponsel serta menjaga kecepatan transfer data seluler, backend Flask kami menerapkan filter penyaringan deteksi tunggal (*Single Bounding Box Filter*). Sistem server secara otomatis menyaring hasil pengolahan data dan hanya merender tepat satu (1) kotak batas dengan tingkat skor kepercayaan (*confidence score*) tertinggi (Argmax) untuk ditampilkan kembali ke layar aplikasi Anda.

Untuk menghargai eksplorasi kuliner yang beragam, hadiah berupa penambahan +50 Poin Kuis hanya diberikan untuk aktivitas pemindaian makanan pertama kali (Unique Scan). Memindai hidangan yang sama (misalnya memindai Tahu Aci berulang kali) tidak akan menambahkan poin kembali ke akun Anda untuk mencegah eksploitasi perolehan skor secara tidak adil.

Kategori C: Peta Budaya & GPS Geofencing

Ketika Anda mengakses halaman detail budaya, aplikasi seluler menggunakan rumus matematika Haversine secara lokal untuk menghitung jarak kelengkungan bumi antara titik koordinat sensor GPS ponsel Anda dengan titik koordinat absolut cagar budaya yang terdaftar di database kami. Jarak ini kemudian diperbarui secara rill di layar Anda.

Pembatasan radius maksimal 100 meter adalah standar presisi yang kami tetapkan untuk memastikan Anda benar-benar melakukan kunjungan fisik rill di area cagar budaya sebelum mengklaim hadiah +100 Poin Kunjungan. Ketika Anda mengetuk tombol "Klaim", koordinat lokasi Anda akan dikirimkan ke server Flask (`/scans/geofence`) untuk divalidasi keabsahannya. Pastikan Anda mengizinkan akses lokasi tingkat "Presisi Tinggi" (*Fine Location*) pada ponsel Anda.

Kategori D: Kuis Budaya, Lencana & Leveling Akun

Setiap kali Anda melakukan aktivitas eksplorasi, server kami akan menambahkan nilai XP (Experience Points) ke profil Anda. Level akun Anda dihitung menggunakan rumus pembagian sisa kelipatan 1.000 XP. Artinya, level akun Anda akan meningkat tepat 1 tingkat setiap kali Anda berhasil mengumpulkan akumulasi kelipatan 1.000 XP (misalnya Level 2 dicapai pada 1.000 XP, Level 3 pada 2.000 XP, dan seterusnya). Anda dapat memantau bar kemajuan level pada tab "Profil".

Poin kuis budaya bernilai +50 Poin hanya ditambahkan ke akun Anda ketika Anda berhasil menjawab kuis tersebut dengan benar untuk pertama kalinya. Setelah kuis berstatus "Selesai" (ditandai dengan warna hijau), Anda masih dapat membuka kuis tersebut untuk mengulas kembali materi sejarahnya, namun menjawab ulang kuis yang sudah selesai tidak akan menambahkan poin tambahan ke akun Anda.

Kategori E: Toko Budaya (UMKM Lokal)

Fitur interaksi UMKM kami menggunakan protokol pengalihan tautan resmi WhatsApp API (`wa.me`). Fitur ini membutuhkan aplikasi WhatsApp resmi terinstal di ponsel Anda. Jika tombol tidak merespon atau menampilkan pesan kesalahan, hal ini umumnya disebabkan karena pemilik toko mendaftarkan format nomor ponsel yang tidak standar di database (misalnya menyertakan tanda hubung, spasi, atau lupa menggunakan kode negara "62"). Laporkan toko tersebut ke tim admin kami jika Anda menemui kendala ini.

Kategori F: Kebijakan Layanan, Pemblokiran & Penghapusan Data

Status ditangguhkan (*suspended*) berarti hak akses masuk akun Anda telah dinonaktifkan secara manual oleh Administrator sistem melalui panel admin utama Flask kami karena adanya indikasi pelanggaran aturan komunitas, laporan ketidaklayakan ulasan, atau aktivitas mencurigakan lainnya yang merugikan integritas data platform. Sanksi ini mengunci seluruh sesi Supabase Auth Anda. Jika Anda merasa penangguhan ini adalah kekeliruan, silakan ajukan klarifikasi melalui email pengembang.
