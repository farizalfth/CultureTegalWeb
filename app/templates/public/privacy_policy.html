Kebijakan Privasi Tegal Culture

Terakhir diperbarui: 15 Juli 2026

Tegal Culture berkomitmen penuh untuk melindungi privasi pengguna kami. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, mentransmisikan, dan melindungi data pribadi Anda saat menggunakan aplikasi seluler dan situs web Tegal Culture.

1. Deklarasi Izin Aplikasi (Application Permissions)

Aplikasi seluler kami meminta izin akses ke beberapa fitur perangkat Anda melalui sistem operasi Android untuk mendukung fungsionalitas fitur interaktif:

Izin Android (Permission) Fungsi dan Alasan Penggunaan (Purpose)
ACCESS_FINE_LOCATION Mengakses lokasi geografis presisi perangkat Anda (GPS) saat aplikasi aktif. Digunakan secara ketat untuk memverifikasi kehadiran fisik Anda di dalam radius 100 meter dari objek cagar budaya guna mengklaim Poin Jelajah (Geofencing Visit Verification) serta memetakan posisi Anda di fitur Peta Budaya interaktif.
ACCESS_COARSE_LOCATION Mengakses perkiraan lokasi perangkat Anda melalui menara seluler atau jaringan Wi-Fi sebagai cadangan verifikasi geofencing jika sinyal GPS presisi tidak tersedia.
CAMERA Mengakses kamera fisik perangkat Anda. Izin ini digunakan secara eksklusif saat Anda membuka fitur Pindai AI Kuliner untuk memotret secara langsung hidangan tradisional khas Tegal guna dianalisis oleh model klasifikasi kecerdasan buatan.
READ_EXTERNAL_STORAGE / READ_MEDIA_IMAGES Mengakses penyimpanan galeri foto pada perangkat Anda. Izin ini digunakan untuk memungkinkan Anda memilih foto kuliner tradisional yang sudah tersimpan sebelumnya di galeri untuk dipindai oleh AI.
POST_NOTIFICATIONS Izin sistem untuk memunculkan pemberitahuan push (Push Notifications) pada perangkat Android 13 ke atas melalui sistem integrasi OneSignal untuk menyampaikan pembaruan info event kebudayaan, pencapaian lencana baru, dan notifikasi kuis.
INTERNET Memungkinkan aplikasi melakukan pertukaran data dua arah dengan server Flask API, modul otentikasi Supabase, dan sistem OneSignal.
ACCESS_NETWORK_STATE Memantau status konektivitas internet perangkat Anda untuk memastikan kestabilan koneksi sebelum mengirim data ke server.

2. Pengumpulan dan Pemrosesan Informasi

  • Informasi Profil Akun: Nama lengkap dan alamat email yang divalidasi secara aman melalui layanan otentikasi Supabase.
  • Media Unggulan (Food Scanner): Foto makanan tradisional yang Anda potret atau pilih dari galeri dikirim secara aman ke server Flask API kami untuk diklasifikasikan secara waktu nyata menggunakan model kecerdasan buatan YOLO26. Foto ini disimpan secara terenkripsi untuk keperluan riwayat pindaian pribadi Anda dan dapat dihapus kapan saja.
  • Data Lokasi: Koordinat geografis Anda hanya dibaca saat Anda menekan tombol "Klaim" atau "Jelajah Sekarang" dan tidak pernah dilacak secara pasif di latar belakang (background location) saat aplikasi ditutup.

3. Keamanan Data & Enkripsi Transit

Kami menerapkan langkah-langkah keamanan tingkat tinggi untuk melindungi data Anda:

  • Enkripsi dalam Perjalanan (Encryption in Transit): Seluruh komunikasi data, termasuk pengunggahan gambar makanan, koordinat lokasi, dan kredensial akun antara aplikasi seluler dan server Flask API kami dilindungi menggunakan enkripsi standar industri tertinggi melalui protokol aman HTTPS (SSL/TLS 1.3) untuk mencegah penyadapan data oleh pihak ketiga di jaringan.
  • Penyimpanan Aman (Storage Security): Sesi otentikasi pengguna di sisi klien disimpan menggunakan enkripsi perangkat keras lokal melalui pustaka Flutter Secure Storage.

4. Hak Penghapusan Data (Data Erasure Rights)

Anda dapat mengakses menu Akun -> Hapus Akun di dalam aplikasi mobile Tegal Culture. Setelah dikonfirmasi, sistem kami akan secara otomatis menghapus seluruh informasi pribadi Anda, riwayat pemindaian makanan, poin kuis, lencana pencapaian, serta kredensial otentikasi Supabase Anda dari server kami secara permanen dan tidak dapat dipulihkan kembali.

5. Hubungi Kami

Jika Anda memiliki pertanyaan mengenai Kebijakan Privasi ini, silakan hubungi tim pengembang kami melalui email resmi di tegalculture.id@gmail.com.
